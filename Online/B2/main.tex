\section{Introduction}

In combinatorics, \textit{Stirling’s number of the second kind \(S(n, k)\)}
   is the number
of ways \underline{partition a set of \( n \) elements into \( k \) non-empty subsets} \([1]\). These
numbers arise  in various areas of mathematics and have applications in 
\textbf{set theory, number theory, and even computer science\footnote{\url{https://en.wikipedia.org/wiki/Stirling_numbers_of_the_second_kind}} }.
\paragraph{}
Stirling’s numbers of the second kind can be defined recursively and have
many interesting properties, which we will explore in this document.


\section{ Properties of Stirling Numbers}

    \subsection{Definition}
    The Stirling number of the second kind, denoted by $S(n,k)$, is defined as the number of ways to divide a set of $n$ elements into $k$ non-empty subsets. It can be written recursively as $S(n,k) = k \cdot S(n-1,k) + S(n-1,k-1)$, for $n > 0$, with the boundary conditions $S(0,0) = 1$, $S(n,0) = 0$ for $n > 0$, and $S(n,k) = 0$ for $k > n$.

    \subsection{Combinational Interpretation}
    Stirling numbers of the second kind have a natural combinatorial interpretation. They count the ways to partition a set of $n$ elements into $k$ non-empty subsets. For example, consider the set $\{1,2,3\}$. The number of ways to partition this set into two subsets is given by $S(3,2) = 3$. These partitions are: 
    \newpage
    \begin{itemize}
        \item \{1\}, \{2, 3\}

    \item \{2\}, \{1, 3\}

 \item \{3\}, \{1, 2\}

    \end{itemize}

    \subsection{Closed Form}
    Stirling numbers of the second kind can be described by the following equation: \\

    \begin{equation}
    S(n, k) = \frac{1}{k!}\sum_{i=0}^{k}(-1)^{k-i}\binom{k}{i}i^{n}=\sum_{i=0}^{k}\frac{(-1)^{k-i}i^{n}}{(k-i)!i!}
\end{equation}
